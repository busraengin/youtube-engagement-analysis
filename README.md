# YouTube Engagement Analysis

## Project Overview
This project explores what drives engagement on YouTube trending videos using Python.  
The analysis focuses on how publishing day, video category, and country influence views and overall performance.

## Dataset
YouTube Trending Videos Dataset (Kaggle)  
40,000+ videos across multiple countries with engagement metrics such as views, likes, and comments.

## Tools Used
- Python
- Pandas
- Data Visualization
- Jupyter Notebook

## Key Insights
- Publishing day significantly affects video performance  
- Education content performs best on Thursday and Sunday  
- Music and Entertainment dominate engagement levels  
- English-speaking countries drive higher visibility  

## How to Run
Open the notebook file in Jupyter Notebook and run all cells to reproduce the analysis.
